Project Ideas

Open-Source Resume Parser
Build a web tool that parses PDF resumes and extracts structured data. Use Python (Flask), JavaScript, and PDF parsing libraries. Focus on ATS compatibility and privacy.
Smart Portfolio Generator
Create a platform for students to generate research portfolios. Integrate with ORCID, allow project uploads, and export to PDF/HTML.
Community Insights Platform
Develop a site where users can share and upvote authentic career advice. Use React for frontend, Node.js for backend, and MongoDB for storage.
HR Contacts Directory
Build a searchable, verified directory of HR contacts for job seekers. Include filters by industry, region, and company size.
AI Interview Coach
An AI-powered web app that simulates interviews, gives real-time feedback, and suggests improvements based on user responses.
Remote Work Readiness Assessment
A tool for companies to assess and improve their remote work policies, including cybersecurity, productivity, and employee wellness.
Automated Research Paper Summarizer
Use NLP to summarize academic papers, highlighting key findings and methodologies for quick review.
Personal Finance Tracker
Build a web app for users to track spending, set budgets, and visualize financial goals using charts and analytics.
Mental Health Support Bot
Create a chatbot that provides mental health resources, daily check-ins, and connects users to professional help.
Sustainable Campus Map
An interactive map for universities showing recycling points, green buildings, and sustainable initiatives.
